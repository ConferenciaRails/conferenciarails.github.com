h1. Entorno para el taller "Testing sin escribir tests..."

Si tenéis pensado asistir al taller de "Testing sin escribir tests con Cucumber y Mundo Pepino" puedes ir preparando tu máquina con el entorno que se utilizará.

Lo primero será instalar la gema de mundo-pepino:

<pre><code>sudo gem install mundo-pepino</code></pre>

Y lo siguiente será crear una aplicación Rails e instalar en la misma InheritedResources como plugin:

<pre><code>rails ev
cd ev
script/plugin install git://github.com/josevalim/inherited_resources.git</code></pre>

Y eso es todo. Si queréis saber que leches significa "ev" no tenéis más remedio que asistir al taller ;)
